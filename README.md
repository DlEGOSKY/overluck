# OVERLUCK

Tower defense roguelite con estética de casino decadente y motor completo en Phaser 3 + TypeScript.

## Meta
Construir primero un MVP jugable y sólido:
- mapa
- oleadas
- enemigos
- torres
- proyectiles
- economía base
- entre oleadas simple

## Stack
- Phaser 3
- TypeScript
- Vite

## Regla
Primero validar:
- loop de combate
- torres básicas
- enemigos básicos
- WaveManager
- ChipManager
- HUD mínimo

Después:
- casino events
- reliquias
- jackpots
- bosses complejos
- polish audiovisual
